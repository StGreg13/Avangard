Техническое задание

0. Общее

Нужно создать локальный git-репозиторий, в который требуется отдельными коммитами сохранять конечный результат выполнения каждого из заданий. То есть закончили выполнение задания - коммит. Порядок выполнения после первого пункта значения не имеет.

Ориентировочное время выполнения: 1-3 часа.

1. Инициализация

Необходимо создать пустой файл с именем .foo в папке с локальным репозиторием.

2. Верстка email-письма

Необходимо из макета сверстать письмо для email-рассылки.

Основное требование: письмо должно корректно отображаться в ПО Microsoft Outlook.

PixelPerfect не требуется.

Макет находится в папке 2-Email.

3. Позиционирование элементов

Выполнив некоторые действия, блоки должны оказаться поверх родительских и соответствовать цветам.

Исходник находится в папке 3-Positioning.

4. AJAX-запрос

На доменное имя api-tz.avangard.local, расположенное по адресу 213.172.24.35, отправить post-запрос, передав следующие данные:

field required comment
foo yes дата в любом формате
bar yes строка
baz no строка длиной от 1 до 255 символов

POST-запрос нужно отправить по адресу http://api-tz.avangard.local/api/test

В случае успеха либо ошибки вывести полученную информацию на экран.

Результат нужно получить в двух видах: с использованием jQuery и с использованием любого JS-фреймворка.

Пример успешного ответа:


[
    {"foo": "2003-01-12 17:38:52", "bar": "Senger-Howe"},
    {"foo": "1993-07-02 05:46:35", "bar": "Beahan PLC"},
    {"foo": "1985-04-16 17:23:10", "bar": "Sauer Ltd"},
    {"foo": "1992-09-02 18:27:01", "bar": "Dibbert Group"}
]

Примеры ошибок:


{
    "error": {
        "code": 405,
        "msg": "Method Not Allowed"
    }
}
            

{
    "error": {
        "code": 400,
        "msg": [
            "The foo is not a valid date.",
            "The bar must be an string."
        ]
    }
}
5. Javascript

На вход подается неассоциативный массив строк из N-элементов.

Задача: средствами javascript vanilla создать общий элемент, внутрь которого нужно добавить все элементы массива, расположенные в своем контейнере.

Пример результата:


<div>
    <p>foo</p>
    <p>bar</p>
    <p>baz</p>
</div>

Сделать то же самое используя библиотеку jQuery.
